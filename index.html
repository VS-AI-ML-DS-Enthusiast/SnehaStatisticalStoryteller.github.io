Turnover Prediction Model
in Python

This project involves creating a machine learning model to predict employee turnover by analyzing various work-related factors. The project begins data preprocessing followed by an exploratory data analysis to identify key factors contributing to turnover. The class imbalance is handled using the SMOTE technique, and k-fold cross-validation is employed to train and evaluate models, ensuring robust performance metrics. The best model is identified and justified based on these metrics. Finally, retention strategies are suggested based on the analysis to help the HR department target employees at risk of leaving.

Loan Default Predictor
in Python

The objective of this project is to build a deep learning model that predicts whether a customer will default using historical data from 2007 to 2015. Predicting loan defaults is critical for companies to manage risk effectively. The dataset, which is highly imbalanced and contains numerous features, presents a significant challenge. The analysis involves data preprocessing, exploratory data analysis, feature engineering, and the development of a deep learning model using Keras with TensorFlow backend to accurately predict the likelihood of loan defaults.
