in SQL

The project analyzes layoff trends across different companies using MySQL by examining various dimensions and metrics. The project involves displaying total layoffs in descending order by company, industry, country, year, and stage, as well as analyzing the progression of layoffs over time. Classification Model

The objective of this project is to build an automatic reporting system that recognizes the boat using deep learning techniques.This project involves building a Convolutional Neural Network to classify boats entering a port. The dataset is split into training and testing sets in an 80:20 ratio and loaded using TensorFlow's image_dataset_from_directory with normalization. The CNN architecture includes convolutional, pooling, and dense layers, and the model is compiled with the Adam optimizer and trained for 20 epochs. The performance is evaluated based on accuracy, precision, and recall metrics.
